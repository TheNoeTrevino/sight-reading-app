GainMultiplier

GainMultiplier: number = 1
  • Defined in Playback/BasicAudioPlayer.ts:29

Multiplicator for gain (volume output). 1 represents the maximum volume in OSMD (100%), but it looks like soundfont-player is louder with volumes > 1. E.g. set osmd.PlaybackManager.audioPlayer.GainMultiplier to 3 if you think the player is too quiet.
